# Учебная практика - Разработка WPF приложения
### Описание:
WPF приложение для управления платежами.

### Основной функционал:
Авторизация с капчей после 3 неудачных попыток <br />
Регистрация пользователей с валидацией пароля <br />
Ролевой доступ: Администратор и Пользователь <br />
Управление данными: пользователи, категории, платежи <br />
Диаграммы для визуализации платежей <br />
Экспорт в Word и Excel <br />
Смена пароля с хешированием SHA1
