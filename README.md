# Учебная практика - Разработка WPF приложения
### Описание:
WPF приложение для управления платежами.

### Основной функционал:
- Авторизация с капчей после 3 неудачных попыток
- Регистрация пользователей с валидацией пароля
- Ролевой доступ: Администратор и Пользователь
- Управление данными: пользователи, категории, платежи
- Диаграммы для визуализации платежей
- Экспорт в Word и Excel
- Смена пароля с хешированием SHA1
